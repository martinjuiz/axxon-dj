# 🎧 Axxon — DJ Presskit

Este repositorio contiene el **presskit oficial de Axxon**, un DJ emergente enfocado en RAW, Hard Groove, Peak, Hard e Industrial Techno.  
El sitio está construido en **HTML estático**, preparado para ser publicado en **GitHub Pages**, e incluye:

- 🎵 Reproductor de **SoundCloud** funcional  
- 🌐 Sistema **multi-idioma (ES/EN)**  
- 📱 Diseño responsive  
- 🖼️ Estructura lista para añadir fotos, enlaces y contenido adicional  

---

## 📄 Características principales

### 🔊 Reproductor de SoundCloud
Incluye un reproductor embebido, fácilmente reemplazable con tu propio set o mix.

### 🌍 Multi-idioma
El sitio permite alternar entre **Español (ES)** e **Inglés (EN)** mediante un conmutador en la cabecera.

### 📱 Responsive
Adaptado para móviles, tablets y escritorio.

### ⚡ Preparado para GitHub Pages
Solo necesitas subir `index.html` a un repositorio público y activar GitHub Pages.

---

## 🚀 Publicación en GitHub Pages

### Opción 1 — Página de usuario  
(Sirve si el repo se llama `usuario.github.io`)

1. Crea un repositorio con el nombre:  
   **`tu-usuario-github.github.io`**
2. Sube el archivo `index.html` (y tus imágenes si las hay).
3. GitHub Pages publicará automáticamente la web en:  
   **https://tu-usuario-github.github.io**

### Opción 2 — Página de proyecto  
(sirve para cualquier repo)

1. Crea un repositorio, por ejemplo:  
   **`axxon-presskit`**
2. Sube el archivo `index.html`.
3. Ve a **Settings → Pages**.
4. En “Build and deployment” selecciona:
   - **Source:** Deploy from a branch  
   - **Branch:** `main`  
   - **Folder:** `/root`
5. La página quedará disponible en:  
   **https://tu-usuario-github.github.io/axxon-presskit**
